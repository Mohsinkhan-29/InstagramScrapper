# Instagram Influencer Analyzer

A Node.js automation tool that extracts and structures 
influencer profile data for marketing campaign analysis.

---

## Tech Stack

**Runtime:** Node.js  
**Automation:** Puppeteer  
**Output:** CSV files

---

## What It Does

- Automates influencer profile data collection
- Extracts engagement metrics and profile information
- Structures data into clean CSV files ready for analysis
- Reduced manual research time from hours → minutes

---

## Use Case

Built for real influencer marketing campaigns — 
when manually reviewing 60-70+ profiles one by one 
was eating up too much time.

---

## Output

Generates structured CSV with:
- Profile data
- Engagement metrics
- Campaign-ready format
